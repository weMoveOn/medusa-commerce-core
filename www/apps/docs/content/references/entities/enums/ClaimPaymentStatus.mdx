# ClaimPaymentStatus

The claim's payment status

## Enumeration Members

### NA

 **NA** = `"na"`

The payment status isn't set, which is typically used when the claim's type is `replace`.

___

### NOT\_REFUNDED

 **NOT\_REFUNDED** = `"not_refunded"`

The payment isn't refunded.

___

### REFUNDED

 **REFUNDED** = `"refunded"`

The payment is refunded.
