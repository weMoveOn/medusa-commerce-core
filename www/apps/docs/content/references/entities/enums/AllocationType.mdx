# AllocationType

The scope that the discount should apply to.

## Enumeration Members

### ITEM

 **ITEM** = `"item"`

The discount should be applied to applicable items in the cart.

___

### TOTAL

 **TOTAL** = `"total"`

The discount should be applied to the checkout total.
