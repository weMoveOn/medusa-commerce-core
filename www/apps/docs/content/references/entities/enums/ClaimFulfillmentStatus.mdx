# ClaimFulfillmentStatus

The claim's fulfillment status.

## Enumeration Members

 **CANCELED** = `"canceled"`

The claim's fulfillments are canceled.

___

 **FULFILLED** = `"fulfilled"`

___

___

### RETURNED

___

### SHIPPED

 **SHIPPED** = `"shipped"`
